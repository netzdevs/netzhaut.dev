Setup

Building Netzhaut requires a few setup procedures to be executed beforehand. First, the source code must be downloaded. Then, the nhmake tool must be build and used to initialize the project.

Linux

          git clone https://github.com/netzdevs/netzhaut.git
          cd netzhaut && ./build/nhmake.sh && ./build/nhmake --init

Build

Before building, please make sure that relevant dependencies are installed. Building is done exclusively using the nhmake tool, which was build during setup. The examples below show how to use nhmake for building, for more info on nhmake usage, go here.

Linux

          ./build/nhmake -b ALL  # build Netzhaut + examples
          ./build/nhmake -b LIBS # build Netzhaut
          ./build/nhmake -b BINS # build examples

Install

Libraries are by default installed to netzhaut/lib, binaries to netzhaut/bin and include files to netzhaut/include.
