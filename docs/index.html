News

2021-06-21 | Dajo Frey
Recent progress includes improved terminal-like features, primarily implemented via the nhtty/nhterm modules. Also, the HTML processing pipeline is now much more flexible with less dependencies between modules, such as nhhtml and nhcss.

2021-05-18 | Dajo Frey
Layout calculation is now done using a separate Fragment-Tree structure compared to the previous hybrid Box/Fragment-Tree. This improves code-readability and eliminates a lot of complexities. Here is a quick look at a Fragment-Tree using the logger: https://imgur.com/a/3GCG64w

2021-05-15 | Dajo Frey
The CSS implementation is now based on compliant W3C CSSOM interfaces which will easily link to the JavaScript engine.

2021-05-11 | Dajo Frey
Added initial implementations for the most common text related properties. Text-Rendering now correctly uses fallbacks. Currently, the only thing completely unsupported regarding text-rendering is emoji-rendering. https://imgur.com/a/3GCG64w

2021-05-05 | Dajo Frey
Margin-collapse is currently disabled but the initial implementations for box-model, inline/block-layout and font-size are done. https://imgur.com/a/vJ09pYe

2021-04-30 | Dajo Frey
Text processing is now done using the common combo FreeType+HarfBuzz. Text rendering is still very much experimental but the general pipeline should be stable now.

2021-04-22 | Dajo Frey
The internal build-system solution used by netzhaut has been moved to a new project, called selfmake.

2021-04-13 | Dajo Frey
