netzhaut | Web Browser Engine

Introduction

netzhaut is an open-source web-browser-engine written from scratch in C99. The main design goals are customizability, conformity and simplicity. The project is currently in early development and thus not yet usable. See the roadmap for more details. The engine is currently being developed for Linux, a Microsoft Windows port is planned once it matures enough.


Build Requirements

For rendering, netzhaut requires Vulkan drivers to be present at runtime.

Linux:

  • GCC
  • X11
  • OpenSSL
  • FreeType

Microsoft Windows:

  • OpenSSL
  • FreeType
