Development Guide

Modules

The implementation is divided into following libraries (or modules):
  • nhcore
  • nhwsi
  • nhtty
  • nhnetwork
  • nhecmascript
  • nhhtml
  • nhwebidl
  • nhencoding
  • nhdom
  • nhgfx
  • nhcss
  • nhurl
This list excludes the netzhaut library since it acts solely as the API, thus providing not much functionality other than a little bit of initialization and dispatching.

Specifications

The following specifications are actively used in development. Implementing a web-browser-engine isn't possible without them. Implementers must use these specifications whenever they are relevant. Also, some specifications are living documents or might become deprecated and implementers need to act accordingly.
